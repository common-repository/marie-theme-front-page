Front-page template for [Marie](https://wordpress.org/themes/marie) theme. Edit sections in the Customizer.

== Description ==

1. Option to show fixed accessibility buttons that alter text size and contrast.
2. About section
3. Feature Cards
4. Edit company/business/personal information (shows in the footer)
5. Show funder/affilliate/client logos
